Offres serveurs

Offre Classique

Pour cette offre, vous pouvez avoir une template de serveur discord, vous n’avez pas de bots sur votre serveur discord. Vous pouvez choisir une police d’écriture disponible sur le site de discord font ou alors ne pas en choisir.

Attention : cette offre ne possède pas de deadline pour la livraison du serveur. Ni configuration de tout type de bot.

Prix de cette offre : 5€

Offre complète

A la différence de l’offre simple, pour cette offre vous pouvez personnaliser votre serveur, vous avez la possibilité de choisir 3 bots de votre choix pour votre serveur discord. Vous pouvez également choisir votre police d’écriture sur le site discord font ou alors ne pas en choisir.

Attention : Cette offre, comme l’offre simple ne possède pas de deadline pour la livraison du serveur. Ni configuration de tout type de bot.

Prix de l’offre : 10€

Offre Premium

Pour cette offre, vous avez à votre disposition tout ce d’ont vous souhaiter, configuration de bots, personnalisation du serveur selon ce que vous souhaitez, police au choix, deadline de votre choix.

Cette offre est prioritaire sur toutes les autres offres disponible.

Prix de l’offre : 25€
